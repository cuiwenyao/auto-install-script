# auto install script 

这是一个自动配置各种应用的脚本合集。

## trojan-auto-install
学习Linux的shell命令使用方法，顺便写一个Trojan自动配置的程序

本脚本只支持Ubuntu20.04及以上版本

请以root身份运行程序

```bash
wget https://raw.githubusercontent.com/cuiwenyao/trojan-auto-install/master/trojan-install.sh && chmod +x trojan-install.sh && ./trojan-install.sh
```
